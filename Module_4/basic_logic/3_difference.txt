//3. What is the difference between OOP and POP?

1.Definition:

oop:OOP stands for Object Oriented Programing.
pop:POP stands for Procedural Oriented Programming.

2.Approach:

oop:    OOP follows bottom up approach.
pop:	   POP follows top down approach.

3.Division:

oop:    A program is divided to objects and their interactions.
pop:    A program is divided into funtions and they interacts.

4.Inheritance supported :

oop:     Inheritance is supported.
pop:     Inheritance is not supported.

5.Access control :

oop:     Access control is supported via access modifiers.
pop:     No access modifiers are supported.

6.Data Hiding :

oop:       Encapsulation is used to hide data.
pop:       No data hiding present. Data is globally accessible.

7.Example :

oop:    C++, Java.
pop:    C, Pascal.
